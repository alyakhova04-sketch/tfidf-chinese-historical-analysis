# TF‑IDF анализ китайских исторических текстов

**Сравнительное исследование ключевых понятий организаций «Красные копья» (红枪会) и «Цинбан» (青帮)**

Проектная работа по курсу «Цифровые технологии как инструмент модернизации исследований иероглифических текстов».  
НИУ ВШЭ, 2026.

---

## Цель исследования

С помощью TF‑IDF анализа выявить и сравнить ключевые слова и слова-маркеры, характеризующие две вооружённые организации первой половины XX века.

## Материалы

- **«Красные копья»** — хроника провинции Шаньдун «近代山东民团研究 (1911–1930)» (около 13 тыс. иероглифов после очистки).
- **«Цинбан»** — газетная заметка «青帮大字张啸林被杀之谜» (около 3 тыс. иероглифов).

## Этапы обработки

1. Очистка от неиероглифических символов (регулярные выражения).
2. Сегментация на слова (библиотека `jieba`).
3. Удаление стоп-слов (список загружен с GitHub, дополнен вручную).
4. TF‑IDF векторизация (`TfidfVectorizer`, 100 наиболее частотных слов).
5. Выделение топ‑20 ключевых слов для каждого текста.
6. Вычисление слов-маркеров (максимальная разница TF‑IDF весов).

## Результаты

- Топ‑20 ключевых слов для «Красных копий» и для «Цинбан».
- Таблица слов-маркеров с числовыми весами.
- Файл `tfidf_result.txt` с сохранёнными результатами.
